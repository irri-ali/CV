# Diensten — AI Sultan

AI Sultan levert AI-gedreven technische diensten voor organisaties die snel iets neer willen zetten zonder een heel team te hoeven inhuren.

---

## 1. E-learning ontwikkeling op maat
Van contentoverzicht naar werkende e-learning in een LMS (Canvas, Brightspace, Moodle).

- LTI 1.3 integratie
- Stap-voor-stap navigatie met voortgangsopslag
- Reflectie-instrumenten (rugzak, journaal, quiz)
- Vimeo-video-integratie met Vimeo Player API
- B1-taalniveau teksten voor zorgprofessionals
- Admin dashboard voor feedback en voortgang

**Geschikt voor:** zorgorganisaties, onderwijsinstellingen, HR-teams

---

## 2. AI-gedreven ontwikkeling (snelle oplevering)
Met Claude Code als werkpaard leveren we in dagen wat normaal weken kost.

- Idee → werkende prototype in één sessie
- Iteratief bouwen met live feedback
- Context bijhouden over meerdere bouwsessies
- Volledige documentatie en geheugenopbouw

**Geschikt voor:** startups, interne tools, tijdgevoelige projecten

---

## 3. Feedbacksysteem & klankbordgroep-ondersteuning
Technische ondersteuning bij co-creatieprocessen met eindgebruikers.

- Feedback verzamelen via live preview overlays
- Gestructureerde feedbackpagina's (gegroepeerd, conflicten zichtbaar)
- Akkoord-systemen met server-side opslag
- Vergadertranscripten verwerken via Fireflies.ai
- Technische besluiten omzetten naar tickets/taken

**Geschikt voor:** productontwikkeling met gebruikerspanels, UX-research

---

## 4. DevOps & server setup
Van nul naar productie op een VPS.

- DigitalOcean / Ubuntu server opzet
- nginx + HTTPS (Let's Encrypt)
- PM2 process management
- GitHub Actions CI/CD pipeline
- Monitoring en revert-procedures

**Geschikt voor:** teams zonder dedicated devops

---

## 5. Bestaand project redden / uitbreiden
Inlezen in een bestaand project en snel doorontwikkelen.

- Code audit en architectuuroverzicht
- Legacy code isoleren zonder te breken
- Nieuwe features naast bestaande functionaliteit
- Migratieplanning (V1 → V2)

**Geschikt voor:** organisaties met tech debt of wisselende freelancers
